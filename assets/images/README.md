Place the project's logo image in this folder and name it `logo.png`.

Recommended sizes:
- 256x256 PNG or SVG for best quality.
- 72x72 or 36x36 (scaled) for nav usage.

Usage:
- The site expects the file at `assets/images/logo.png`.
- If you prefer a different filename, update the `src` attributes in the header anchors.

Accessibility:
- Keep the `alt` text on the `<img>` (e.g. `alt="JSTP logo"`).
